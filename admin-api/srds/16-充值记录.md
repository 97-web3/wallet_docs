## F-ADMIN-034: 获取充值记录

**描述**: 获取充值记录列表。

**HTTP 方法**: `GET /api/v1/admin/ledger/deposits`

**请求参数**:
```
?page=1
&page_size=20
&network=
&currency=
&user_id=
&status=                     // pending, confirming, completed, failed
&min_amount=
&max_amount=
&created_from=
&created_to=
```
