## F-ADMIN-085: 配置余额阈值

**描述**: 配置 Vault 余额告警阈值。

**HTTP 方法**: `PUT /api/v1/admin/vault/thresholds`

**请求 Schema**:
```json
{
  "network": "Ethereum",
  "chain_id": 1,
  "currency": "ETH",
  "thresholds": {
    "low": "100.000000",
    "critical": "50.000000"
  },
  "notifications": {
    "email_enabled": true,
    "email_recipients": ["ops@company.com", "finance@company.com"],
    "webhook_enabled": true,
    "webhook_url": "https://hooks.example.com/vault-alert"
  }
}
```

**响应 Schema**:
```json
{
  "success": true,
  "message": "阈值配置已更新",
  "data": {
    "network": "Ethereum",
    "currency": "ETH",
    "thresholds": {
      "low": "100.000000",
      "critical": "50.000000"
    },
    "updated_at": "2025-12-10T10:00:00Z",
    "updated_by": "admin@company.com"
  }
}
```

**权限要求**:
- 角色: Admin

## F-ADMIN-086: 手动同步 Vault 余额

**描述**: 手动触发 Vault 余额同步。

**HTTP 方法**: `POST /api/v1/admin/vault/{chain_id}/sync`

**路径参数**:
- `chain_id`: 链 ID

**响应 Schema**:
```json
{
  "success": true,
  "message": "同步任务已启动",
  "data": {
    "network": "Ethereum",
    "sync_id": "sync-20251210-001",
    "status": "syncing",
    "started_at": "2025-12-10T10:00:00Z"
  }
}
```

**权限要求**:
- 角色: Admin
