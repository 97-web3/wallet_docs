## 管理员角色枚举（建议模板）

> 实际角色/权限以“角色权限管理”接口返回为准；本节提供企业后台常见角色模板，便于与 Admin Web 文档对齐。

| 角色 | 代码 | 描述 |
|------|------|------|
| 超级管理员 | super_admin | 全部权限，含紧急开关与双人控制 |
| 系统管理员 | admin | 日常运营配置、风控治理与全局查看 |
| 财务人员 | finance | 出金审批、转账执行、对账与报表 |
| 人力 | hr | 员工目录与薪资批次创建/审批 |
| 运维/风控 | ops | 告警治理、系统状态与渠道异常处理 |
| 审计员 | auditor | 全量只读 + 导出（受控） |
