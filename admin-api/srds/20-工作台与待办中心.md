该模块用于把“待审批/待处理/待复核”的事项统一收口，减少跨页面跳转成本。

## F-ADMIN-137: 获取工作台待办事项

**描述**: 获取工作台待办事项列表，支持按类型/状态/优先级/归属筛选。

**HTTP 方法**: `GET /api/v1/admin/workbench/items`

**请求参数**:
```
?page=1
&page_size=20
&type=                       // withdrawal_approval, payroll_approval, vault_adjustment_approval, batch_execution, reconcile_diff, alert
&status=                     // pending, in_progress, blocked, done
&priority=                   // low, medium, high, critical
&mine=                       // true/false（仅看我的事项）
```

**响应 Schema**:
```json
{
  "success": true,
  "data": {
    "items": [
      {
        "id": "wi-20251210-001",
        "type": "withdrawal_approval",
        "title": "待审批提现申请 #W20251210001",
        "priority": "high",
        "status": "pending",
        "owner": "admin-10001",
        "created_at": "2025-12-10T10:00:00Z",
        "due_at": "2025-12-10T12:00:00Z",
        "link": "/admin/withdrawals/W20251210001",
        "summary": "5000 USDT · Ethereum"
      }
    ]
  },
  "pagination": {
    "page": 1,
    "page_size": 20,
    "total": 8,
    "total_pages": 1
  }
}
```

**权限要求**:
- 角色: Admin, Finance, HR, Ops（按事项类型细分）
