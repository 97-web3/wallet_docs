# 系统设置

> 返回 [后台管理前端目录](./README.md)

## 页面概述

- **页面路径**: `/admin/settings`
- **权限要求**: Admin 角色
- **主要功能**: 角色权限管理、系统配置、告警设置、审计日志、费率与限额配置

#### 1.3 编辑管理员

**触发方式**: 点击管理员行的「编辑」按钮

**可编辑内容**:
- 姓名
- 分配角色
- 邮箱
- 状态（启用/禁用）

**注意**: 登录账号不可修改

#### 1.4 重置密码

**触发方式**: 点击管理员行的「更多」→「重置密码」

**重置方式**:
| 方式 | 说明 |
|------|------|
| 发送重置链接 | 发送密码重置链接到邮箱 |
| 设置临时密码 | 手动设置临时密码，首次登录需修改 |

#### 1.5 禁用/启用管理员

**触发方式**: 点击管理员行的「禁用」或「启用」按钮

**禁用后效果**:
- 该账号无法登录
- 已有的登录会话立即失效
- 保留历史操作记录

### 2. 角色权限管理

**功能描述**: 管理系统角色和权限配置

**用户角色**: Admin

**页面路径**: `/admin/settings/roles`

#### 1.1 角色列表

**UI 组件**:
- **Table**: 角色列表表格
- **Tag**: 角色类型标签
- **Button**: 添加、编辑按钮

**表格列**:
| 列名 | 字段 | 说明 |
|------|------|------|
| 角色名称 | name | 角色标识 |
| 显示名称 | display_name | 中文名称 |
| 描述 | description | 角色说明 |
| 用户数 | user_count | 关联用户数量 |
| 类型 | type | 系统/自定义 |
| 状态 | status | 启用/禁用 |
| 操作 | - | 查看、编辑、删除 |

**角色说明**:

> **重要**: 系统支持动态创建角色。管理员可根据业务需要自由创建角色，并为每个角色分配不同的权限组合，然后将角色授权给管理员账号。

**系统预置角色（示例模板）**:
| 角色 | 说明 | 可修改 |
|------|------|--------|
| admin | 系统管理员，拥有全部权限 | 不可删除，权限不可修改 |

#### 2.3 完整权限模块定义

系统共定义 11 个功能模块，每个模块包含细分权限项：

| 模块ID | 模块名称 | 权限项 |
|--------|----------|--------|
| dashboard | Dashboard | 查看、导出 |
| vault | Vault资金管理 | 查看、充值、提现、配置预警、查看历史 |
| transfer | 转账管理 | 查看、创建批次、编辑批次、删除批次、执行转账、查看详情、导入Excel |
| web2users | Web2用户管理 | 查看、创建用户、编辑用户、删除用户、冻结/解冻、导出数据 |
| web3users | Web3用户管理 | 查看、编辑用户、设备管理、标记黑名单、冻结/解冻、导出数据 |
| currencies | 币种管理 | 查看、添加币种、编辑币种、启用/禁用、删除币种 |
| swap | Swap配置 | 查看、DEX管理、汇率配置、执行Swap |
| withdrawal | 出金审批 | 查看、审批、查看详情、导出记录 |
| roles | 角色权限管理 | 查看、创建角色、编辑角色、删除角色、分配角色 |
| flow | 资金流水记录 | 查看、查看详情、导出记录、高级筛选 |
| blacklist | 黑名单地址管理 | 查看、添加地址、编辑地址、删除地址、监测管理、批量导入、导出列表 |

| 类型 | 说明 | 计算方式 |
|------|------|----------|
| 固定费用 | 固定金额 | 手续费 = 固定值 |
| 百分比 | 按比例收取 | 手续费 = 金额 × 费率 |
| 动态Gas | 根据 Gas 计算 | 手续费 = Gas × 系数 |

### 6. 审计日志

**功能描述**: 查看系统操作审计日志

**用户角色**: Admin（完整访问）/ Finance（只读）

**页面路径**: `/admin/settings` → 审计日志

#### 6.1 审计日志列表

**UI 组件**:
- **Table**: 日志列表
- **DatePicker**: 日期筛选
- **Select**: 操作类型筛选

**筛选条件**:
| 字段 | 组件 | 说明 |
|------|------|------|
| 日期范围 | RangePicker | 操作时间 |
| 操作人 | Select | 筛选操作人 |
| 操作类型 | Select | 用户管理/审批/配置... |
| 操作对象 | Input | 对象 ID 搜索 |

**审计日志表格**:
| 列名 | 字段 | 说明 |
|------|------|------|
| 时间 | created_at | 操作时间 |
| 操作人 | operator | 操作者信息 |
| 操作类型 | action_type | 操作分类 |
| 操作描述 | description | 操作说明 |
| 对象类型 | target_type | 操作对象类型 |
| 对象ID | target_id | 操作对象标识 |
| IP 地址 | client_ip | 客户端 IP |
| 详情 | - | 查看详情 |

**操作类型分类**:
| 类型 | 说明 |
|------|------|
| USER_FREEZE | 用户冻结 |
| USER_UNFREEZE | 用户解冻 |
| USER_TERMINATE | 离职处理 |
| WITHDRAWAL_APPROVE | 提现通过 |
| WITHDRAWAL_REJECT | 提现驳回 |
| TRANSFER_CREATE | 创建转账批次 |
| TRANSFER_EXECUTE | 执行转账 |
| VAULT_ADJUST | Vault 调整 |
| CONFIG_UPDATE | 配置修改 |
| ROLE_UPDATE | 角色权限修改 |

#### 6.2 日志保留策略

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 在线保留期 | 1 年 | 可在线查询的日志保留时间 |
| 归档保留期 | 3 年 | 归档后的日志保留时间 |
| 归档周期 | 每月 | 自动归档历史日志的周期 |
| 导出格式 | Excel/CSV | 支持的导出格式 |

> **合规说明**: 审计日志默认保留 1 年供在线查询，超过 1 年的日志将自动归档。归档日志可按需申请恢复查询。如有特殊合规要求，可调整保留策略。
