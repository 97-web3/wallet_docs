# 数据库连接 (db/)

## F-DB-001: GORM 初始化

- **描述**: GORM 数据库连接初始化
- **文件**: `common/db/gorm.go`
- **功能**:
  - MySQL/MariaDB 连接
  - 连接池配置
  - 慢查询日志
  - 自定义日志集成 (go-zero logx)

---

## 配置项

| 参数 | 说明 | 默认值 |
|------|------|--------|
| Host | 数据库主机 | - |
| Port | 端口 | 3306 |
| Username | 用户名 | - |
| Password | 密码 | - |
| Database | 数据库名 | - |
| MaxIdleConns | 最大空闲连接 | - |
| MaxOpenConns | 最大打开连接 | - |
| ConnMaxLifetime | 连接最大生命周期 (秒) | - |
| SlowThreshold | 慢查询阈值 (毫秒) | 0 |
| LogLevel | 日志级别 | 1 |
