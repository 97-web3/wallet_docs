## F-SIGN-012: 签名消息

**描述**: 对消息进行签名（用于链下验证）。

**gRPC 方法**: `SignMessage`

**请求 Schema**:
```json
{
  "request_id": "msg_20251210_001",
  "seed_id": "seed_hot_001",
  "chain": "ETH",
  "derivation_path": "m/44'/60'/1'/0/10001",
  "message": "Hello, World!",
  "requester": "business_service"
}
```

**响应 Schema**:
```json
{
  "code": 0,
  "message": "签名成功",
  "request_id": "msg_20251210_001",
  "signature": "0x1b2c3d4e5f...",
  "signed_at": "2025-12-10T10:00:00Z"
}
```

**使用场景**:

| 场景 | 描述 |
|------|------|
| 身份验证 | 验证地址所有权 |
| 链下授权 | 签署链下授权消息 |
| 数据签名 | 对任意数据签名 |

**签名格式**:

- **ETH**: 使用 `eth_sign` 格式，包含 `\x19Ethereum Signed Message:\n` 前缀
- **TRON**: 使用 TRON 标准消息签名格式
- **BSC**: 与 ETH 相同
