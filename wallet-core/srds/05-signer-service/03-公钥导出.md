# 公钥导出

> 返回 [Signer 服务目录](./README.md)

---

## F-SIGN-008: 导出单个公钥

**描述**: 导出指定派生路径的公钥。

**gRPC 方法**: `ExportPublicKey`

**请求 Schema**:
```json
{
  "seed_id": "seed_hot_001",
  "chain": "TRN",
  "derivation_path": "m/44'/195'/0'/0/10001",
  "requester": "chainrpc_service"
}
```

**响应 Schema**:
```json
{
  "code": 0,
  "message": "",
  "seed_id": "seed_hot_001",
  "chain": "TRN",
  "derivation_path": "m/44'/195'/0'/0/10001",
  "public_key": "04a1b2c3d4e5f6...",      // 未压缩公钥 (hex)
  "compressed_pubkey": "02a1b2c3d4..."    // 压缩公钥 (hex)
}
```

## F-SIGN-009: 批量导出公钥

**描述**: 批量导出多个派生路径的公钥。

**gRPC 方法**: `ExportPublicKeyBatch`

**请求 Schema**:
```json
{
  "seed_id": "seed_hot_001",
  "chain": "TRN",
  "start_index": 0,
  "count": 100,
  "account": 0,
  "change": 0,
  "requester": "chainrpc_service"
}
```

**响应 Schema**:
```json
{
  "code": 0,
  "message": "",
  "seed_id": "seed_hot_001",
  "chain": "TRN",
  "start_index": 0,
  "count": 100,
  "public_keys": [
    {
      "index": 0,
      "derivation_path": "m/44'/195'/0'/0/0",
      "public_key": "04...",
      "compressed_pubkey": "02..."
    }
  ]
}
```
