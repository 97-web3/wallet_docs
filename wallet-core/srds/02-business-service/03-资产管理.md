## F-BUS-009: 获取资产概览

**描述**: 获取用户所有资产的汇总信息。

**gRPC 方法**: `GetAssetOverview`

**请求 Schema**:
```json
{
  "symbol": "USDT",    // 可选：指定币种
  "price": ""          // 可选：指定价格
}
```

**响应 Schema**:
```json
{
  "success": true,
  "total_valuation_usdt": "10000.00",
  "items": [
    {
      "asset": "USDT",
      "total_balance": "5000.00",
      "available_balance": "4500.00",
      "frozen_balance": "500.00",
      "valuation_usdt": "5000.00"
    },
    {
      "asset": "TRX",
      "total_balance": "10000.00",
      "available_balance": "10000.00",
      "frozen_balance": "0",
      "valuation_usdt": "1000.00"
    }
  ]
}
```

## F-BUS-010: 近期交易列表

**描述**: 获取最近的交易记录，用于首页展示。

**gRPC 方法**: `ListRecentTransactions`

**请求 Schema**:
```json
{
  "limit": 10,           // 返回条数，默认 3
  "asset": ["USDT"]      // 可选：过滤资产
}
```

**响应 Schema**:
```json
{
  "success": true,
  "items": [
    {
      "id": "123456",
      "type": "deposit",
      "asset": "USDT",
      "amount": "100.00",
      "timestamp": 1702300800,
      "status": "confirmed",
      "memo": "充值"
    }
  ]
}
```

## F-BUS-011: 余额变动记录

**描述**: 获取详细的余额变动历史记录。

**gRPC 方法**: `ListBalanceRecords`

**请求 Schema**:
```json
{
  "page": 1,
  "page_size": 20,
  "asset": "USDT",        // 可选：过滤资产
  "type": "deposit"       // 可选：记录类型
}
```

**响应 Schema**:
```json
{
  "success": true,
  "total": 100,
  "items": [
    {
      "id": "123456",
      "type": "deposit",
      "asset": "USDT",
      "amount": "100.00",
      "timestamp": 1702300800,
      "status": "confirmed",
      "memo": "TRC20 充值"
    }
  ]
}
```
