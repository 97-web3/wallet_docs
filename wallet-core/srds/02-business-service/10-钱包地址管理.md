## F-BUS-034: 获取钱包地址列表

**描述**: 获取用户保存的提现地址列表。

**gRPC 方法**: `ListWalletAddresses`

**请求 Schema**:
```json
{
  "asset": "USDT",     // 可选
  "chain": "TRN"       // 可选
}
```

**响应 Schema**:
```json
{
  "success": true,
  "items": [
    {
      "asset": "USDT",
      "chain": "TRN",
      "address": "TKvjY...",
      "label": "我的波场钱包",
      "is_default": true,
      "created_at": 1702300800
    }
  ]
}
```

---

## F-BUS-035: 添加钱包地址

**描述**: 添加新的提现地址。

**gRPC 方法**: `AddWalletAddress`

**请求 Schema**:
```json
{
  "asset": "USDT",
  "chain": "TRN",
  "address": "TKvjY...",
  "label": "我的波场钱包",
  "set_default": true
}
```

**响应 Schema**:
```json
{
  "success": true,
  "message": "地址添加成功"
}
```
