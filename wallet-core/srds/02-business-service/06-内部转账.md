## F-BUS-016: 获取转账信息

**描述**: 获取内部转账相关信息。

**gRPC 方法**: `GetTransferInfo`

**请求 Schema**:
```json
{
  "asset": "USDT",              // 可选：指定资产
  "recipient": "10002"          // 可选：接收方标识
}
```

**响应 Schema**:
```json
{
  "success": true,
  "assets": [...],
  "available": "4500.00",
  "recipient_uid": "10002",
  "recipient_nickname": "接收方昵称",
  "tips": "内部转账免手续费，实时到账"
}
```

## F-BUS-017: 创建内部转账

**描述**: 执行内部用户间转账。

**gRPC 方法**: `CreateTransfer`

**请求 Schema**:
```json
{
  "recipient_uid": "10002",
  "asset": "USDT",
  "amount": "100",
  "memo": "转账备注"
}
```

**响应 Schema**:
```json
{
  "success": true,
  "message": "转账成功",
  "transfer_id": "T123456"
}
```
