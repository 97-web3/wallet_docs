## F-BUS-026: 获取通知概览

**描述**: 获取未读消息数量。

**gRPC 方法**: `GetNotificationSummary`

**请求 Schema**:
```json
{}
```

**响应 Schema**:
```json
{
  "success": true,
  "unread_count": 5
}
```

---

## F-BUS-027: 获取通知列表

**描述**: 分页获取通知消息列表。

**gRPC 方法**: `ListNotifications`

**请求 Schema**:
```json
{
  "page": 1,
  "page_size": 20,
  "only_unread": false
}
```

**响应 Schema**:
```json
{
  "success": true,
  "total": 50,
  "unread_count": 5,
  "items": [
    {
      "id": "N123456",
      "type": "deposit",
      "title": "充值到账",
      "content": "您的 100 USDT 充值已到账",
      "timestamp": 1702300800,
      "read": false
    }
  ]
}
```
