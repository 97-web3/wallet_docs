## F-BUS-012: 获取充值信息

**描述**: 获取支持的充值资产和网络信息。

**gRPC 方法**: `GetDepositInfo`

**请求 Schema**:
```json
{
  "asset": "USDT",    // 可选：指定资产
  "query": ""         // 可选：搜索关键词
}
```

**响应 Schema**:
```json
{
  "success": true,
  "assets": [
    {
      "asset": "USDT",
      "total_balance": "5000.00",
      "available_balance": "4500.00",
      "frozen_balance": "500.00",
      "valuation_usdt": "5000.00"
    }
  ],
  "networks": [
    {
      "chain": "TRN",
      "fee": "1",
      "eta": "1-5分钟",
      "min_amount": "1",
      "per_order_limit": "100000"
    },
    {
      "chain": "BSC",
      "fee": "0.5",
      "eta": "3-10分钟",
      "min_amount": "1",
      "per_order_limit": "100000"
    }
  ]
}
```

## F-BUS-013: 创建充值地址

**描述**: 为用户生成指定资产和网络的充值地址。

**gRPC 方法**: `CreateDeposit`

**请求 Schema**:
```json
{
  "asset": "USDT",
  "chain": "TRN"       // 网络：TRN/ETH/BSC
}
```

**响应 Schema**:
```json
{
  "success": true,
  "address": "TKvjY...",
  "memo_tag": "",                    // 如需 Memo/Tag
  "qrcode_url": "https://...",       // 二维码地址
  "asset": "USDT",
  "chain": "TRC20",
  "min_deposit": "最小充值 1 USDT",
  "tips": "请确认网络正确，否则可能导致资产丢失"
}
```
