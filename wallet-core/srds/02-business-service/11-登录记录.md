## F-BUS-036: 获取登录记录

**描述**: 获取用户的登录历史记录。

**gRPC 方法**: `ListLoginRecords`

**请求 Schema**:
```json
{
  "page": 1,
  "page_size": 20
}
```

**响应 Schema**:
```json
{
  "success": true,
  "total": 100,
  "items": [
    {
      "device_name": "iPhone 15 Pro",
      "platform": 2,          // 1=Web, 2=iOS, 3=Android, 4=API
      "ip_address": "192.168.1.1",
      "location": "北京市",
      "login_time": 1702300800,
      "result": "success",
      "is_current": true,
      "is_trusted": true
    }
  ]
}
```
