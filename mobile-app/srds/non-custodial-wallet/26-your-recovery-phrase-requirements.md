# 您的恢复短语页 - 软件需求规格说明书

## 1. 页面标识
- **页面名称**: 您的恢复短语页
- **页面ID**: your-recovery-phrase
- **页面描述/用途**: 允许用户查看已创建钱包的恢复短语（助记词），提供复制功能，并显示安全警告

## 2. UI组件
### 2.1 布局概述
页面采用垂直布局，顶部为返回按钮和标题，中间为安全警告卡片、助记词显示区域，底部为复制按钮和完成按钮。

### 2.2 组件详情
| 组件名称 | 组件类型 | 位置 | 描述 |
|---------|---------|------|------|
| 返回按钮 | 按钮（图标） | 顶部左侧 | 返回上一页面 |
| 页面标题 | 文本标签 | 顶部中央 | "您的恢复短语" |
| 安全警告卡片 | 信息卡片 | 标题下方 | 黄色背景卡片，包含警告文字 |
| 警告文字 | 文本段落 | 警告卡片内 | 红色警告文字 |
| 助记词显示区域 | 容器视图 | 中部 | 深色圆角矩形，包含12个助记词 |
| 助记词卡片 | 文本标签（12个） | 助记词区域内 | 编号+单词格式，如"1 merge"、"2 margin" |
| 复制到剪贴板按钮 | 次要操作按钮 | 助记词区域下方 | 图标+文字，复制助记词 |
| 完成按钮 | 主要操作按钮 | 底部 | 荧光绿色背景，关闭页面 |

## 3. 功能需求
### 3.1 主要功能
- 显示钱包的12个助记词
- 提供安全警告提示
- 允许用户一键复制助记词到剪贴板
- 提供返回功能

### 3.2 业务逻辑
1. **页面展示**：
   - 从加密存储读取助记词
   - 以编号列表形式显示
   - 显示安全警告

2. **安全警告内容**：
   - "请勿分享您的恢复短语！"
   - "如果有人获得了您的恢复短语，他们就能够窃取您的资金。"

3. **复制功能**：
   - 点击复制按钮
   - 将12个助记词复制到剪贴板
   - 显示复制成功提示
   - 设置剪贴板超时清理（安全考虑）

4. **完成操作**：
   - 点击完成按钮
   - 返回上一页面或设置页面

## 4. 用户交互
### 4.1 用户操作
| 操作 | 触发条件 | 响应 |
|-----|---------|------|
| 点击返回按钮 | 用户点击左上角返回图标 | 返回到上一页面（通常是设置页） |
| 点击复制按钮 | 用户点击"复制到剪贴板"按钮 | 复制助记词，显示成功提示 |
| 点击完成按钮 | 用户点击底部"完成"按钮 | 返回到上一页面 |

### 4.2 导航流程
```
您的恢复短语页 → 点击返回 → 设置页/管理资料页
您的恢复短语页 → 点击完成 → 设置页/管理资料页
```

### 4.3 手势操作
- 支持垂直滚动
- 长按助记词单词可能显示复制选项（可选）

## 5. 数据需求
### 5.1 数据展示
| 数据字段 | 数据类型 | 描述 |
|---------|---------|------|
| 页面标题 | 字符串 | "您的恢复短语" |
| 警告文字 | 字符串 | 安全警告内容 |
| 助记词列表 | 对象数组 | 12个助记词及其序号 |
| 复制按钮文字 | 字符串 | "复制到剪贴板" |
| 完成按钮文字 | 字符串 | "完成" |

### 5.2 数据输入
本页面无用户输入

### 5.3 数据来源
- 助记词：从本地加密存储读取
- 需要用户身份验证（Face ID/指纹）才能查看

## 6. 验证规则
### 6.1 输入验证
- 进入页面前需要身份验证（Face ID/Touch ID/密码）

### 6.2 错误处理
| 错误场景 | 错误信息 | 处理方式 |
|---------|---------|---------|
| 助记词读取失败 | "无法加载恢复短语" | 显示错误，提供重试 |
| 身份验证失败 | "验证失败" | 拒绝显示，返回上一页 |
| 复制失败 | "复制失败，请重试" | 显示提示 |

## 7. 技术考量
### 7.1 状态管理
- 助记词数据（加密）
- 身份验证状态
- 复制状态

### 7.2 API集成
- 生物识别验证（Face ID/Touch ID）
- 加密存储读取
- 剪贴板API

### 7.3 安全考量
1. **身份验证**：
   - 进入页面前必须通过生物识别或密码验证
   - 验证失败不显示助记词

2. **显示保护**：
   - 设置FLAG_SECURE防止截图
   - 应用切换到后台时自动隐藏内容

3. **剪贴板安全**：
   - 复制后设置超时自动清除（如60秒）
   - 提示用户剪贴板风险

4. **数据保护**：
   - 助记词在内存中加密
   - 页面销毁时清理数据
