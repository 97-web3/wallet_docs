# 备份助记词页2（明文状态）- 软件需求规格说明书

## 1. 页面标识
- **页面名称**: 备份助记词页2（明文状态）
- **页面ID**: backup-mnemonic-2
- **页面描述/用途**: 以明文显示生成的12个助记词，供用户抄写备份，是备份助记词页1点击"显示助记词"后的状态

## 2. UI组件
### 2.1 布局概述
页面布局与备份助记词页1相同，但助记词区域显示为明文，模糊效果和禁止图标被移除，显示12个编号的助记词单词。

### 2.2 组件详情
| 组件名称 | 组件类型 | 位置 | 描述 |
|---------|---------|------|------|
| 返回按钮 | 按钮（图标） | 顶部左侧 | 返回上一页面 |
| 页面标题 | 文本标签 | 顶部中央偏左 | "备份助记词" |
| 安全提示文字 | 文本段落 | 标题下方 | 多行警告文字，强调截图风险 |
| 助记词标签 | 文本标签 | 提示下方 | "助记词（12个单词）" |
| 显示助记词按钮 | 图标按钮 | 助记词标签右侧 | 眼睛图标配合文字"显示助记词"（当前显示状态） |
| 助记词显示区域 | 容器视图 | 中部 | 深色圆角矩形区域，以网格布局显示12个助记词 |
| 助记词卡片 | 文本标签（12个） | 助记词区域内 | 每个包含序号和单词，如"1 foster"、"2 model" |
| 验证备份按钮 | 主要操作按钮 | 底部上方 | 荧光绿色背景，跳转到验证页面 |
| 稍后再备份链接 | 文本链接按钮 | 底部 | 荧光绿色文字，跳过备份流程 |

## 3. 功能需求
### 3.1 主要功能
- 以明文显示12个助记词
- 提供清晰的编号和单词排列
- 允许用户轻松抄写助记词
- 支持切换回模糊状态
- 提供验证备份功能
- 持续显示安全警告

### 3.2 业务逻辑
1. **助记词显示**：
   - 12个助记词以3列4行的网格布局显示
   - 每个助记词包含序号（1-12）和英文单词
   - 示例：1 foster, 2 model, 3 simple, 4 guitar, 5 satisfy, 6 poem, 7 celery, 8 wasp, 9 eyebrow, 10 gas, 11 innocent, 12 peanut
   - 使用清晰易读的字体和适当的间距

2. **显示/隐藏控制**：
   - 当前处于显示状态
   - 点击"显示助记词"按钮可切换回模糊状态
   - 按钮图标和文字相应更新

3. **安全提示**：
   - 保持与页1相同的安全警告
   - "切勿截屏"文字使用醒目颜色

4. **用户操作流程**：
   - 用户查看明文助记词
   - 使用笔纸按顺序抄写
   - 妥善保管抄写的助记词
   - 点击"验证备份"进入验证环节
   - 或选择"稍后再备份"

## 4. 用户交互
### 4.1 用户操作
| 操作 | 触发条件 | 响应 |
|-----|---------|------|
| 点击返回按钮 | 用户点击左上角返回图标 | 返回到上一页面，提示未完成备份 |
| 点击显示助记词按钮 | 用户点击眼睛图标或文字 | 切换到模糊状态（备份助记词页1） |
| 长按助记词单词 | 用户长按某个助记词 | 可选：显示复制选项或无操作（出于安全考虑） |
| 点击验证备份按钮 | 用户点击底部"验证备份"按钮 | 导航到助记词验证页面 |
| 点击稍后再备份 | 用户点击底部"稍后再备份"链接 | 跳过备份流程，标记为未备份状态 |

### 4.2 导航流程
```
备份助记词页2 → 点击返回 → 上一页面
备份助记词页2 → 点击显示助记词 → 切换到备份助记词页1（模糊状态）
备份助记词页2 → 点击验证备份 → 助记词验证页
备份助记词页2 → 点击稍后再备份 → 主页（未备份状态）
```

### 4.3 手势操作
- 支持垂直滚动查看完整内容
- 建议禁用助记词的复制功能（安全考虑）
- 支持系统返回手势

本页面无输入验证需求

## 7. 技术考量
### 7.1 状态管理
需要管理的状态：
- 助记词数据（12个单词）
- 助记词显示/隐藏状态（当前为显示）
- 用户是否已查看助记词
- 用户在此页面的停留时间
- 截图检测状态

### 7.2 API集成
- **BIP39库**：验证助记词有效性
- **本地存储**：临时保存助记词（加密）
- **截图检测**：监听系统截图事件
- **剪贴板控制**：防止助记词被复制
- **分析统计**：记录用户行为

### 7.3 安全考量
1. **截图保护**：
   - 设置FLAG_SECURE防止截图（Android）
   - 监听截图事件（iOS/Android）
   - 检测到截图时立即隐藏助记词
   - 显示安全警告弹窗
   - 或在截图中模糊助记词内容

2. **显示保护**：
   - 设置超时自动隐藏（可选）
   - 应用切换到后台时自动隐藏
   - 返回前台需重新验证（可选）

3. **复制保护**：
   - 禁用助记词文本的选择功能
   - 禁用长按复制菜单
   - 清除剪贴板中的敏感内容

4. **内存安全**：
   - 助记词在内存中加密存储
   - 页面销毁时立即清理数据
   - 不在日志中记录助记词

5. **UI安全**：
   - 使用不透明背景防止透视
   - 字体大小适中，易读但不过大
   - 持续显示安全警告

6. **用户引导**：
   - 明确指示用户使用笔纸抄写
   - 警告不要截图或复制
   - 强调助记词的重要性
