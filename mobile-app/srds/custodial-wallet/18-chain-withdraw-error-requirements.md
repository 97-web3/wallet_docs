## 1. 页面标识

| 属性 | 值 |
|------|-----|
| 页面ID | CW-018 |
| 页面名称 | 链上提币详情页面（地址错误提示） |
| 版本 | 1.0 |
| 更新日期 | 2025-12-08 |
| 状态 | 草稿 |

### 功能描述
与CW-017相同的页面，但展示了地址验证失败时的错误提示状态。

---

## 2. 说明

此页面与CW-017（链上提币详情页面）为同一页面，区别在于展示了地址验证错误的状态。

### 错误状态展示
- 提币地址输入框下方显示红色"地址错误"提示
- 错误提示清晰明显
- 用户需要修正地址后才能继续

---

## 3. 功能需求

### FR-CW-018-01: 地址错误提示

| 字段 | 内容 |
|------|------|
| **ID** | FR-CW-018-01 |
| **描述** | 当用户输入的地址格式错误时显示错误提示 |
| **验收标准** | 1. **假设** 用户输入无效地址, **当** 验证失败, **则** 在地址输入框下方显示红色"地址错误"文字<br>2. **假设** 显示错误提示, **当** 渲染, **则** 使用红色字体突出显示<br>3. **假设** 用户修正地址, **当** 验证通过, **则** 自动清除错误提示 |
| **优先级** | 必须 |

### FR-CW-018-02: 阻止提交

| 字段 | 内容 |
|------|------|
| **ID** | FR-CW-018-02 |
| **描述** | 地址错误时阻止用户继续提交 |
| **验收标准** | 1. **假设** 地址验证失败, **当** 用户点击"下一步", **则** 不允许提交并保持在当前页面<br>2. **假设** 尝试提交, **当** 触发, **则** 可选择性震动输入框或显示Toast提示 |
| **优先级** | 必须 |
