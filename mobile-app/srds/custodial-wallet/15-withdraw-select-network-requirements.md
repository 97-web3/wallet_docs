# 提币选择网络页面 - 软件需求规格说明书

## 1. 页面标识

| 属性 | 值 |
|------|-----|
| 页面ID | CW-015 |
| 页面名称 | 提币选择网络页面 |
| 版本 | 1.1 |
| 更新日期 | 2025-12-08 |
| 状态 | 草稿 |

### 功能描述
用户选择提币使用的区块链网络，显示每个网络的到账时间和最小充币金额，与充币选择网络页面功能相同。

---

## 2. 说明

此页面与CW-008（选择网络页面）功能基本相同，仅在业务场景上有所区别（充币 vs 提币）。

### 主要功能
- 显示支持的网络列表
- 显示重要安全提示
- 显示每个网络的到账时间和最小金额
- 选择网络后进入提币详情页

---

## 3. 参考

请参考：`08-select-network-requirements.md`

### 差异点
- 业务场景：提币而非充币
- 导航流程：选择后进入提币详情页（输入地址）而非充币详情页

---

## 4. 提币特定功能需求

### FR-CW-015-01: 提币手续费显示

| 字段 | 内容 |
|------|------|
| **ID** | FR-CW-015-01 |
| **描述** | 在网络选择列表中显示每个网络的提币手续费 |
| **验收标准** | 1. **假设** 显示网络列表，**当** 渲染每个网络项，**则** 显示该网络的提币手续费（如"手续费: 1 USDT"）<br>2. **假设** 手续费不同，**当** 对比显示，**则** 用户可以直观比较各网络费用差异<br>3. **假设** 手续费变动，**当** 显示，**则** 显示实时手续费并标注"实时"标签 |
| **优先级** | **必须** |

### FR-CW-015-02: 网络手续费对比

| 字段 | 内容 |
|------|------|
| **ID** | FR-CW-015-02 |
| **描述** | 帮助用户选择最经济的网络 |
| **验收标准** | 1. **假设** 显示网络列表，**当** 有多个网络，**则** 按手续费从低到高排序（可切换）<br>2. **假设** 最低手续费网络，**当** 显示，**则** 标注绿色"最低费用"标签<br>3. **假设** 用户点击排序切换，**当** 触发，**则** 可在"推荐排序"和"费用排序"之间切换 |
| **优先级** | 应该 |

### FR-CW-015-03: 提币到账时间显示

| 字段 | 内容 |
|------|------|
| **ID** | FR-CW-015-03 |
| **描述** | 显示预估的提币到账时间 |
| **验收标准** | 1. **假设** 显示网络项，**当** 渲染，**则** 显示预估到账时间范围（如"5-30分钟"）<br>2. **假设** 网络拥堵，**当** 显示时间，**则** 显示延长后的预估时间并标注"拥堵"<br>3. **假设** 时间差异大，**当** 对比，**则** 最快到账网络标注"最快"标签 |
| **优先级** | 应该 |

### FR-CW-015-04: 最小提币金额显示

| 字段 | 内容 |
|------|------|
| **ID** | FR-CW-015-04 |
| **描述** | 显示每个网络的最小提币金额要求 |
| **验收标准** | 1. **假设** 显示网络项，**当** 渲染，**则** 显示该网络的最小提币金额<br>2. **假设** 用户余额低于某网络最小提币金额，**当** 显示该网络，**则** 禁用该网络选项并显示"余额不足"提示 |
| **优先级** | 必须 |

## 5. 验收标准汇总

1. ✅ 网络列表完整显示
2. ✅ 每个网络信息准确
3. ✅ 重要提示清晰显示
4. ✅ 点击网络正确跳转到提币详情页
5. ✅ UI符合设计稿
6. ✅ **每个网络显示提币手续费**
7. ✅ **最低手续费网络标注"最低费用"标签**
8. ✅ **显示预估到账时间范围**
9. ✅ **余额不足时禁用对应网络选项**
10. ✅ **支持排序切换（推荐/费用）**

---

## 6. 无障碍访问要求

- [ ] 所有文本颜色对比度符合WCAG 2.1 AA标准（最低4.5:1）
- [ ] 按钮和可交互元素最小触摸目标为44x44pt
- [ ] 支持屏幕阅读器，所有UI元素有适当的无障碍标签
- [ ] 网络列表项有明确的role="listitem"标记
- [ ] 禁用状态的网络项通过aria-disabled标识
- [ ] 手续费和时间信息通过VoiceOver朗读
- [ ] 排序切换状态通过无障碍公告通知用户
- [ ] 支持键盘导航（适用于平板/桌面设备）
