# 提币选择币种页面 - 软件需求规格说明书

## 1. 页面标识

| 属性 | 值 |
|------|-----|
| 页面ID | CW-011 |
| 页面名称 | 提币选择币种页面 |
| 版本 | 1.0 |
| 更新日期 | 2025-12-08 |
| 状态 | 草稿 |

### 功能描述
用户选择要提币的加密货币币种，显示每个币种的可用余额，支持搜索和热门币种推荐。

## 3. 功能需求

### FR-CW-011-01: 币种列表展示

| 字段 | 内容 |
|------|------|
| **ID** | FR-CW-011-01 |
| **描述** | 显示所有支持提币的币种及可用余额 |
| **验收标准** | 1. **假设** 进入页面, **当** 加载完成, **则** 显示所有支持提币的币种<br>2. **假设** 每个币种项, **当** 显示, **则** 包含币种信息和可用余额<br>3. **假设** 点击某个币种, **当** 触发点击, **则** 跳转到选择提币方式页面 |
| **优先级** | 必须 |

### FR-CW-011-02: 搜索功能

| 字段 | 内容 |
|------|------|
| **ID** | FR-CW-011-02 |
| **描述** | 用户可以搜索币种 |
| **验收标准** | 1. **假设** 用户输入币种名称, **当** 输入内容, **则** 实时过滤匹配的币种<br>2. **假设** 搜索无结果, **当** 无匹配项, **则** 显示"未找到相关币种" |
| **优先级** | 必须 |

### FR-CW-011-03: 热门币种推荐

| 字段 | 内容 |
|------|------|
| **ID** | FR-CW-011-03 |
| **描述** | 显示热门币种快捷入口 |
| **验收标准** | 1. **假设** 显示热门推荐, **当** 加载, **则** 显示最常用的币种胶囊按钮<br>2. **假设** 点击推荐币种, **当** 触发, **则** 直接进入该币种的选择提币方式页面 |
| **优先级** | 应该 |

### FR-CW-011-04: 可用余额显示

| 字段 | 内容 |
|------|------|
| **ID** | FR-CW-011-04 |
| **描述** | 显示每个币种的可用余额 |
| **验收标准** | 1. **假设** 显示余额, **当** 渲染, **则** 使用千位分隔符<br>2. **假设** 余额为0, **当** 显示, **则** 可选择性置灰或标注"余额不足" |
| **优先级** | 必须 |

## 4. 用户交互

### 4.1 用户工作流
1. 用户从主页或币种详情页点击"提现"
2. 进入选择币种页面
3. 可以通过搜索或浏览列表选择币种
4. 查看每个币种的可用余额
5. 选择有余额的币种
6. 进入选择提币方式页面

### 4.2 导航流程
- 主页/币种详情页 → 提币选择币种页面
- 提币选择币种页面 → 选择提币方式页面

## 6. 验证规则

- 搜索支持币种名称和全称匹配
- 搜索不区分大小写
- 余额使用千位分隔符
- 余额精度根据币种类型调整
